How it works

The workbook structure

  • Cover — rounds processed, dates, totals
  • Summary — per-PDF comment counts per round
  • One tab per language — e.g. CZ, deDE, nlBE, with autoFilter
  • __data — tool-managed; do not delete (round history)

How rounds are matched

When you upload a workbook from a previous round, each new comment is matched to a prior one using both location and text similarity. Matched: appended to the existing thread. Unmatched and not a reply to a prior comment: flagged in red as a late comment.

Filtering scales

Each language tab has a Scale column. Click the filter arrow on the header row in Excel to show only CSSRS Baseline, only qb-PRO, etc. Sort by any column the same way.

Adding more rounds later

When Round 2 arrives: open this tool, drop the new Round 2 RWS zip into Step I, optionally drop this round's screens zip into Step II, drop your Round 1 workbook into Step III, click Run. The tool merges everything into a single updated workbook.

Privacy. Files never leave your computer. All PDF parsing, image cropping, and Excel building happens inside this browser tab.
